> My Favorite Albums Software Documentation Introduction

This is the landing page for the My Favorite Albums Software Documentation, written by Aidan Bartlett, as the final assignment for ENGL 396. 
On the left side of the browser, there are several tabs linking readers to different pages of documentation. The documentation is broken up for 
two primary audiences: 

1. __Music enthusiast with some coding experience who wants to publish their own data to My Favorite Albums__
2. __End user who wants to view, use, and understand the My Favorite Albums site on the web__

Each audience has:
- Conceptual Documentation
- Task Documentation
- Reference Documentation
